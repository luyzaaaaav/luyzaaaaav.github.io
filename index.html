Mas antes, vamos entender como publicar uma página no GitHub pages!

Primeiramente, você precisa clicar em "settings", depois disso feito, é necessário clicar em "pages", agora você precisa se certificar que você já fez um clone de sua página (para isso, entre no git Bash e digite "git clone" e coloque o link do repositório que você deseja clonar). Agora é necessáro deixar na opção "GitHub Actions", depois clicar em configurar como html estático e, por fim, clicar em "commit changes"
